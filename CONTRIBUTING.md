### 🤝 **Contributing**  

We welcome contributions to make **ChatGPT Desktop Client** even better! Here's how you can contribute:  

1. **Report Issues**:  
   - Found a bug or have a feature request? Open an issue on our [GitHub Issues page](https://github.com/MK-986123/chatgpt-desktop-client/issues).  

2. **Fork the Repository**:  
   - Clone the project to your local machine:  
     ```bash
     git clone https://github.com/MK-986123/chatgpt-desktop-client.git
     cd chatgpt-desktop-client
     ```  

3. **Set Up Development Environment**:
   - Install dependencies:
     ```bash
     npm install
     ```
   - Start development server:
     ```bash
     npm start
     ```

4. **Make Changes**:  
   - Work on a new feature or fix a bug.  
   - Follow the project's code style and guidelines.
   - Test your changes thoroughly on Linux (Ubuntu 24.04+ preferred)

5. **Test Builds**:
   - Test different package formats:
     ```bash
     npm run dist:appimage  # Test AppImage build
     npm run dist:snap      # Test Snap build  
     npm run dist:deb       # Test DEB build
     ```

6. **Submit a Pull Request**:  
   - Push your changes to a new branch and open a pull request on the main repository.  
   - Include a clear description of changes and any testing performed

7. **Join the Discussion**:  
   - Engage with the community, provide feedback, and help others with their issues.  

---

### 🌟 **Ways to Help**  
- Test the app on different Linux distributions (especially Ubuntu 24.04/25.04)
- Report OAuth authentication issues or improvements
- Suggest new keyboard shortcuts or UI enhancements
- Test different package formats (AppImage, Snap, DEB)
- Improve accessibility features
- Contribute to documentation and help guides
- Share feedback on performance and desktop integration

### 🔧 **Development Guidelines**
- Use modern ES6+ JavaScript features
- Follow existing code style and structure
- Test OAuth flows thoroughly
- Ensure changes work across different Linux desktop environments
- Validate that builds work for all supported package formats
- Update documentation when adding new features

### 🐛 **Bug Reports**
When reporting bugs, please include:
- Linux distribution and version
- Package format used (Snap, AppImage, DEB)
- Steps to reproduce the issue
- Expected vs. actual behavior
- Console logs if applicable

Thank you for your contributions and support! Together, we can build an amazing ChatGPT desktop experience.
